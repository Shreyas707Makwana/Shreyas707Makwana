## 🚀 What I've Been Cooking Lately

### 🎯 **[DocSense AI](https://ai-powered-personal-agent-platform.vercel.app/)** - *The Document Whisperer*
> Ever wanted to have a conversation with a PDF? Now you can. This RAG-powered platform processes **10MB documents in 6-15 seconds** and remembers **25+ conversation turns**. It's like ChatGPT, but for your documents, and it actually cites its sources with **92% accuracy**.

**The magic sauce:** *Next.js + FastAPI + Vector Databases + A sprinkle of AI wizardry*

### 🗣️ **[Neural Text-to-Speech](https://huggingface.co/spaces/Shreyas707Makwana/Speech_Synthesis)** - *Teaching Machines to Sing*
> Built a TTS system that sounds so human, it scored **4.2/5 on MOS evaluation**. Used Tacotron2 + HiFi-GAN and optimized it with CUDA to synthesize speech **45% faster**. Because why should Siri have all the fun?

### 💬 **[Sync Pad](https://github.com/Shreyas707Makwana/Sync_Pad)** - *Google Docs, But Make It Real-time*
> Real-time collaborative editor that handles **50+ users simultaneously** with **sub-150ms latency**. Built with CRDT algorithms because I believe in the power of collaboration (and WebSockets).
